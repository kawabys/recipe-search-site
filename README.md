# レシピ検索サイト

## 概要
PHPとMySQLで構築したレシピ検索・管理サイトです。  
ユーザーはレシピの検索・閲覧・追加ができ、管理者は専用画面でレシピや管理者アカウントの管理が可能です。

---

## 主要機能
- レシピの一覧表示・詳細表示
- キーワード検索機能
- 材料によりAND絞り込み
- レシピの追加機能
- 管理者用画面
  - レシピの編集・削除
  - 管理者アカウントの追加・削除
- 画像アップロード対応

---

## 技術スタック
- PHP (PDO使用)
- MySQL
- HTML / CSS / JavaScript
- フロントエンドはシンプルで見やすく設計

---

## ディレクトリ構成
/admin/ # 管理画面用PHPファイル（管理者アカウント管理含む）
/css/ # CSSスタイルシート
/js/ # JavaScriptファイル
/uploads/ # 画像アップロード用フォルダ（Git管理外推奨）
db.php # データベース接続ファイル（サンプル）
index.php # トップページ
list.php # レシピ一覧ページ
add.php # レシピ追加ページ
recipe.php # レシピ詳細ページ
